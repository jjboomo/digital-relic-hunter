**2. 目标用户自相矛盾**
```
真正懂文言文的人 → 不用编程
真正写代码的人 → 不懂文言文
两个圈子的交集 → 约等于空集
```

**3. 技术护城河为零**
- 词法分析器几百行

**4. 商业价值论证失败**
- 文旅噱头？政府项目能拿但不可持续

# kusti8/proton-native 尸检报告

**项目定位：** 用 React 写原生桌面 app，号称 Electron 杀手

**死亡时间：** 约 2019 年，commit 停更

---

## 🪦 尸检报告

### 死亡诊断书

```
诊断结果: 先天性心脏骤停 + 后天营养不良
致命病因:
├── 1. 依赖链断裂 (libui-node 比你先死)
├── 2. 技术选型失误 (Node.js + Qt 性能优势没兑现)
├── 3. 生态真空 (npm 生态全靠 Electron，轮子一个没有)
└── 4. 作者体力透支 (一个人硬撑，根本扛不住跨平台适配)
```

### 死因详解

**① 底层依赖暴毙**

```
proton-native
    └── libui-node (死了)
            └── libui (半死不活)
```

当你引以为傲的底层库集体扑街，你还拿什么呼吸？

**② 性能牛皮吹破**

声称比 Electron 轻 90%，实际：
- 冷启动还是要跑 Node.js
- Qt 层吃掉不少内存
- 开发者体验稀烂（API 改一次炸一次）

**③ 错过了窗口期**

2017 年启动的时候 Electron 已经起飞，等你打磨好，2020 年 Tauri 都出来了。

**④ 一个人扛不住跨平台**

Windows/macOS/Linux 三端适配、Bug 修复、文档维护、Issue 回复——这是 Team 的活，不是个人项目能干的。

### 墓志铭

> "一个理想主义者的轻量化梦想，死于生态建设的缺失和一个人战斗的孤独。"

# Gridea 项目尸检报告

## 📋 基本信息

| 项目 | Gridea |
|------|--------|
| GitHub Stars | 11.2k |

| 状态 | 半死不活 |

## 一、尸检报告

### 🔍 死因分析

**1. 市场时机悲剧**

- 错过了2018-2020年知识付费风口
- 等它成熟时，Notion/Obsidian/Remnote 已经吃掉了"第二大脑"市场
- Typora 靠 Markdown 编辑器场景稳住了，Gridea 定位不上不下

**2. 定位模糊，爹不疼娘不爱**

- 说它是博客客户端？GitHub Pages 生态太小众
- 说它是写作工具？Notion/Obsidian 功能碾压它
- 说它是 CMS？WordPress/Ghost 才是正经方案
- **结果：四不像，谁都不是必需品**

**3. 技术债堆积**
```
Electron + 老旧 Vue 2 + 陈旧依赖
→ 安装包越来越大 (100MB+)
→ 启动慢，占内存
→ 现代用户没有耐心等
```

**4. 社区运营拉胯**
- 文档更新停滞
- issue 无人回应
- 主题/插件生态几乎为零
- 开发者独狼模式，没有社区飞轮

**5. 商业化无解**
- 开源 + 免费 + 没捐赠渠道
- 不收钱就养不起人
- 收钱就有人骂

---

## 二、秽土转生方案

### 🎯 重新定位：面向普通人的"轻量级个人网站生成器"

**核心逻辑改变：**
```
旧：技术人员的博客工具
新：普通人的个人主页/作品集/简历网站
```

### 🛠 执行方案

**Phase 1: 技术重构 (6个月)**

```python
1. Electron → Tauri (Rust后端)

   - 启动速度: 8秒 → 1秒

2. Vue 2 → Vue 3 + Vite
   - 开发体验现代化
   - 减小内存占用

3. 增加 AI 辅助功能
   - AI 写作助手（基于本地模型）
   - AI 生成博客配图
   - AI SEO 建议
```

**Phase 2: 功能重定义 (3个月)**

```javascript
// 核心差异点
{
  "目标用户": "不会代码的普通人",
  "核心场景": [
    "个人简历网站",
    "作品集展示",
    "个人品牌主页",
    "简单的博客"
  ],
  "差异化": "一键部署 + AI 辅助 + 好看模板"
}
```

**Phase 3: 社区激活 (持续)**


### 💡 关键转折点

> **不要做"更好的 Gridea"，要做"普通人的 Squarespace/Notion 替代"**
